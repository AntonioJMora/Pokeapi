# PokeApi

## Descripción del proyecto
- Para el html y el css apenas he usado IA, lo he creado yo y ya luego le he añadido las funcionalidades mediante JS
- Para el JS si que he usado algo más de IA y la API que te comenté que tenía ya hecha, ya que se me ha complicado algo más el tema de poner en pantalla lo que me daba y el tema de cada tipo, pero se ha conseguido solucionar y se entiende el código, pues hasta que no lo he entendido no he dejado de preguntarle qué hacía o cómo.

## Partes
### Partes de HTML
El proyecto cuenta con 2 partes
- index.html = Que es donde se encuentra el buscador y donde aparecen los elementos a buscar
- detalle.html = En la cual, al pulsar sobre un elemento en el index, nos lleva a esta, modificando su información con JS.

### Parte de CSS
El proyecto cuenta con un solo css, que abarca ambos HTML y las clases que se añaden mediante JS.

### Parte de JS
El proyecto tiene 2 JS:
- api.js = Se encarga de obtener los datos de la pokeapi
- ui.js = Se encarga de escuchar y de dar los datos que se muestran en pantalla
